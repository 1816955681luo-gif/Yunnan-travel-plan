# Yunnan-travel-plan
# 🏔️ 云南旅游规划助手

一个帮助你规划云南旅行的交互式网站，支持行程管理、倒计时提醒和待办事项清单。

## ✨ 功能特色
- **行程规划**：可视化行程安排
- **倒计时**：距离出发时间提醒
- **待办清单**：管理旅行前的准备事项
- **数据保存**：本地存储，无需登录

## 🌐 在线访问
**网站地址**：https://1816955681luo-gif.github.io/Yunnan-travel-plan/

## 🛠️ 技术栈
- HTML5 / CSS3 / JavaScript
- 本地存储 (LocalStorage)
- GitHub Pages 部署

## 📝 使用说明
1. 打开网站链接
2. 设置你的出发日期
3. 添加行程和待办事项
4. 数据会自动保存到浏览器
